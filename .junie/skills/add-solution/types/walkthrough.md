# Type: Walkthroughs

## Format

```markdown
* Blog post [(:lang:)] on [Blog Name](Blog Link): [Title](Link) (🧃`vX.x`)
```

## Details

-   **Language**: If the article is not in English, add the language code in parentheses, e.g., `(:de:)`, `(:es:)`.
-   **Blog/Publication Name**: The name of the hosting blog or site.
-   **Title**: The title of the walkthrough.
-   **Version (🧃)**: The version of Juice Shop the walkthrough was written for.

## Example

```markdown
* Blog post on [IncognitJoe](https://incognitjoe.github.io/): [Hacking(and automating!) the OWASP Juice Shop](https://incognitjoe.github.io/hacking-the-juice-shop.html) (🧃`v2.x`)
```
