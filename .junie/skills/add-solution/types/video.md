# Type: Hacking Videos

## Format

```markdown
* [Title](Link) by [Author](Link) (🧃`vX.x`)
```

If the entry is part of a series or playlist:

```markdown
* [Series/Channel Name](Link) (🧃`vX.x`)
  * [Episode Title](Link)
```

## Details

-   **Title**: The title of the video or video series.
-   **Author**: The creator of the video or the channel name.
-   **Version (🧃)**: The version of Juice Shop the video was recorded for.
-   **Language**: If the video is not in English, add the language code in parentheses, e.g., `(:de:)`, `(:es:)`.

## Example

```markdown
* [OWASP Juice Shop | TryHackMe Burp Suite Fundamentals](https://youtu.be/6n1pI9dJpW4) by [CyberInsight](https://www.youtube.com/channel/UCmJJUewPWfnyzvZRrFHlykA)
```
