# Type: Scripts & Tools

## Format

```markdown
* [Tool Name](Tool Link) written in [Language] by [@Author](Author Link) (🧃`vX.x`)
```

## Details

-   **Tool Name**: The name of the tool or script.
-   **Language**: The programming language the tool was written in (e.g., Python, JavaScript).
-   **Author**: The username or real name of the author.
-   **Version (🧃)**: The version of Juice Shop the tool was tested with.

## Example

```markdown
* [Automated solving script for the OWASP Juice Shop](https://github.com/incognitjoe/juice-shop-solver) written in Python by [@incognitjoe](https://github.com/incognitjoe) (🧃`v2.x`)
```
